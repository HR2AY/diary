# 框架

我是一个外表木讷的人，但我的脑中对感受到的风吹草动都极为敏感。我自测为ADHD。

我是一个想让世界更好的人。我想让每一个人都变得更自由。我是一个正在学习经济学的本科生，我思考了很多，我可以说接收了高于常人水平的信息。

我在两年的日日夜夜中挣扎着寻求（失眠的截图），终于在这一个晚上找到了答案。

我刷视频的时候，突然有了一种被剥夺感。

我构建了一个模型，或者说一个逻辑框架，用于解释快乐，幸福，注意力，AI，工业革命，自由，我还认为我想通了当下的我们现在应该做什么。

我恰巧发现这个理论可以很好的放在唯物史观的框架中，甚至能额外加入心理学、神经科学、经济学的佐证，能解释很多当下社会涌现的问题、进行中的争论。

我把这个模型愚昧地称为“用AI解放注意力的第四次工业革命”。我自己想表达的最终意思是“注意力”在未来会变成类似“劳动力”、“信息”这种资源，AI会因为解放注意力而引起下一次工业革命。

我的这个想法是由网上见到的很多博主的想法结合而成，我的想法还比较稚嫩、漏洞百出；但大家看完视频能获得对当下有点帮助的领悟，我的视频便达到目的了。

## **一。从碎片的记忆，拼出模型的拼图**

我最近看到了一个理论，描述了人们主观感到的时间流逝速度成类ln型收敛。它让我感受到，也许我们眼中的世界并不是按时间均匀分布；那我们到底通过什么来界定自己的存在呢？

这个理论立足于一个很强的假设模型：回忆。网上人们都说18年后的生活没有以前好了：我们都觉得以前的生活更快乐。这是为什么？我在自我分析时，能感觉到“回忆”占了一部分原因。我们在回忆中能产生与当下经历水平接近的多巴胺/内啡肽反应（有医学证实）。这种物质构成了我们快乐的来源。正好我最近看过一个硅谷华人工程师的B站采访，他说“我们生活中一件很重要的事情就是管理多巴胺”：他如此强调，让我坚信不止我自己，普通人的生活中，快乐也可能占很重要的地位，只是国内互联网还没有形成共识。

我建立了一个脆弱的假设：我们经常（在信念不足时，或处于平常生活的无意识中）会将快乐作为感受自己存在的一个指标，做的很多事情都是在追求这些快乐物质，很多设定好的目标，都是在一次次的内啡肽正反馈中建立起来的（这与目标的实际意义没什么关系，仅是站在神经学层面而言）。

我观察了一下人们对幸福的追求，以及幸福的人们生活中在做什么，我发现他们当中有很多都展现出明显的多巴胺低阈值，他们能从很多小事中获取快乐的正反馈，而这种状态不是秘密，世界各地的文明在历史中早已洞悉，并引导其子民来达到这种状态。

那这些和AI，工业革命有什么关联呢？我们引入另一个概念：注意力。这个已经生活化的词越来越多被提及，主要是在人工智能/机器学习领域(Attention is all you need)。注意力，第一次用在描述人类等有智慧的生物以外的其他对象上。我们的精神世界是由注意活动构建而成的，我们与世界的交互是消耗注意力的结果。

**【注意】：如果比作一个函数，它由人的感觉输入，经过了意识的处理，最终输出到两个方向：**

**一是通往外界，形成了“反应”；二是通向内心，形成了“记忆”。**

**注意是如此的基础且重要，而现在出现了我们可以控制的注意力自动化“机器”，那个隐约的想法可能已经隐约在你脑中成型了；我会之后大致说一下我的看法；不过我想先讨论一下“注意力”，与“快乐”可能有着紧密的联系。**

## 二。直观的感受：注意力机制主导着多巴胺，和我的生活

我回忆了这一年的大学生活，有哪些事情曾让我快乐：客观地讲，打游戏，刷视频，听故事，吃东西，打球……而且我会经常在梦中模拟这些场景。有一个显著的现象：当在经历这些事时，快乐总是发生在结果大于等于我们预期的时候。在梦中尤为明显：我们的梦中会经常出现无法预期（比如无尽的场景）或者不及预期（如被心爱的人抛弃），因为梦境通常会体现出未解的情绪冲突或欲望，也就是快乐的缺失部分。我想说的是，预期的实现与否和快乐的感受有着很强的关联，就像一个动态的标准：正因如此，人们说增加幸福的一种实现路径就是降低期待。

而预期与注意力的关系则更为密切，甚至可以称为“相伴而生”：我们作为人类，在脑资源有限的情况下是不可能接受到所有信息的；而所谓“注意”，可以说就是将“外界信息”按照【期望】的方向【投注】在【意识】中：没有”期望“，我们会失去主动”接收信息“的理由。这很好理解，比如你对某领域没任何了解也不想了解，你就不会记住这一领域的任何知识（哪怕你听了一天的水课）。

以上便构建起了一个简易的逻辑：我们作为人是一定有期望的（可以转化成后面讨论的需求），这个期望是源自我们内心的，它引导我们获取信息，对比后产生快乐。人的主观能动取得成功，情绪上我们也获得了奖励。这便是我解析的自己所作所为的注意力的底层机制。如果我们能按照这个循环进行下去，结果便是理想中的人生该有的样子。

但是就如之前提到的，为什么我们现在越来越少感到快乐了呢？随着历史发展，我们不应该朝着更完善的福祉发展，而特别是近几年，我们越来越焦虑、越来越分神呢？抛开宏观因素，我们自己的生活中是不是遗漏了什么细小的致命威胁？相信很多人能顺着思路想到：我们的“注意力机制”，很可能掌握在他人的手中；我们希望构建的”注意力稳态“，已经被外界打破。

## 三。上升到周围：我们的快乐在的摇摇欲坠的注意力支配平台上

重新提起让我快乐的事：里面的”打游戏，刷视频“是不是有很熟悉的感觉？没错，它们是最明显的、掌握着我们一天空闲中有限注意力资源的平台。它们的控制力显然超越了人们最初的设计，在大学中尤为明显：不仅是所谓“迷茫”的大学生无意识间在每天的生活中消耗了大量注意力，连那些我知道的“有目标且努力”甚至已经有成果的人也没能逃过在注意力垄断平台上消磨时光。这个现象显然和人之间传统“成功”层面的划分没有必然的因果关系了。

我们再来想想现在我们所羡慕的过去可能有哪些特征。过去的时代：长视频平台，热门水平基于全站的播放量等总数据，智能推荐还未普及；社交平台：较简单的基于人际的推荐，还记得“漂流瓶”吧，除开隐私安全因素，其推荐机制完全没任何“注意力/用户习惯”的设计；游戏：网游之外，单机游戏还没有失去热度，尤其是我们记忆中的“小时候”，单机可能超过了网游的占比，而单机的注意力消耗上限是可以期望的。从“注意力”的视角来看，当时的我们没有遇到如今的垄断平台，可以将注意力用在成长、用在看世界里。我们怀念过去的自己，有一部分就是在怀念过去”注意力自由“的时代。

那为什么现在的我们是如此无法自拔地对平台”上瘾“呢？之前提到，注意力的底层机制带来了多巴胺等快乐物质；而”上瘾“的成因正是多巴胺层面的行为路径依赖。连成一条因果线：

注意力被平台牵引 → 按照设定的路径获取多巴胺 → 多巴胺在高频触发中不断分泌 → 大脑形成了对“即时反馈”的依赖路径 → 其他的深度活动（学习、观察、思考）因反馈延迟而显得乏味 → 我们逐渐丧失对“延迟满足”的耐受力 → 于是，无意识中我们便会回到能快速获得快感的平台 → 注意力被进一步榨取与集中 → 形成稳定的“注意力垄断”结构 → 快乐因同质化被稀释，留下空虚与疲惫。

这一条链条的尽头，不是更幸福的自我，而是平台的控制与被动中追逐快乐的无力：我们现在的快乐建立在注意力支配平台上。

## 四。成瘾、AI；算法早已改造了平台：思考能被取代吗？

因果链的第一条：注意力被平台牵引，这便是改变的肇始。现在的平台在竞争中基本都引入了注意力控制机制。举两个例子：

短视频推荐机制简要版：

①用户行为采集：平台会提取你对某个视频的反应（如留存时长、点赞评论）

②用户特征建模：基于采集数据，系统生成该用户的兴趣向量，即用户在不同视频的可能停留率

③排序、推荐：按照向量生成推荐，进入循环，强化向量

用户正反馈来源：视频本身带来的多巴胺奖励；刷视频的“超预期”带来的奖励效应

三角洲游戏机制简要版（源自网络）：

①：隐藏分机制：通过游戏内行为模式、停留时长，生成隐藏分，即玩家画像

②：收集机制：通过玩家画像动态调整对局内搜集物资的价值区间，与高价值物资爆率强相关

③：pvp机制：玩家根据画像匹配对手，通过消耗物资实现pvp，击败对方获取对方物资并实现收益

玩家正反馈来源：收集高价值物资的超预期奖励；击败玩家的多层次(自我认可、预期、社交)奖励

所以说从某种角度而言，二者都是通过最大化用户停留度，形成用户社区；短视频便可进行规模商业化（如借助用户向量优化广告服务），游戏平台也可售卖付费道具（用户也有意愿购买）。

这是正常的商业逻辑，对服务商而言是情理之中的；我在这里是想抛出新的观点：这个模式浪费了用户大量“注意力”，对双方都不利；AI的出现，能瓦解这套注意力控制模式；服务商也有可能通过AI实现商业模式的转化，双方都能进行优化。

先来说说为什么存在“注意力浪费”：

很简单，我们作为用户，在进行“刷视频”、“玩游戏”时，大部分时间是不会使用“思维”或者说是【无意识】的，而这个时候，既对我们来说在消耗【注意力】，又对平台而言是没价值的，因为用户是在【有意识】的时候进行付费活动的。我们常说的“上瘾”，便是因为我们在【有意识】的自我反省时，发现我们【无意识】消耗了大量注意力，便十分后悔。

那么【无意识】的这一部分对双方，是不是都是副作用的呢？对于用户：花费大量不必要的“注意力”在活动中，变得麻木，没有产生记忆，感到后悔；对于平台：浪费额外资源，同时降低了用户付费意愿和离开的风险。我想，这就是浪费——算法制造了大量无效刺激，用户制造了大量无效反应。

然后我尝试解释一下为什么我觉得AI能优化这一部分：

直觉上我们的目标便是减少【无意识】，并保持双方原反馈尽量不变：AI不同于传统算法的地方在于它能动态理解**用户的认知状态**。如果有一天，出现了能量化用户【无意识】状态的算法，引入短视频/游戏的机制，用户在陷入【无意识】时突然从视频/游戏中感觉到自己的状态，这便唤醒了【意识】；这便实现了【优化】。

这和之前的算法不就是多了一个标签吗？我认为这个改动用代码模拟了作为人类的用户，第一次深入到【意识】层面（之前的行为标签只是代表着表层的刺激-反应系统）；这就是构成AI定义之一：模拟意识。人的“思考”不会被AI替代，但“不思考”可以。而且我认为，技术层面并不难，这可能是不久的将来便可实现的AI的优化。

## 五。意识与幸福

先阐明前面提到的【有意识】和【无意识】，我提出这两个变量，还能解释【注意力】与【幸福】的潜在联系：之前我们曾简单地假设过，幸福=获取快乐，这里进一步补充，幸福是人类在【有意识】的状态下对一件事获取到的“快乐”与它得到的结果的价值进行衡量之后的结果。举个控制变量的例子：比如我们在不同的情境下吃东西：

①你奖励自己去吃一家想了很久的餐厅。

- 你知道自己为什么吃（有意识）；
- 食物让你产生味觉愉悦（快乐）；
- 你觉得这是合理的犒劳，有意义（价值）。

你在“有意识”的注意力下体验快乐，这个快乐被理性整合，转化为幸福。

②你在刷视频时连续吃了几包零食。

- 你没有计划，只是条件反射地去吃（无意识）；
- 吃的瞬间确实有味觉愉悦（快乐）；
- 事后你觉得浪费、口干、后悔（价值为负）。

虽然你同样“快乐”，但因为是在**无意识状态下**、缺乏价值整合，这段体验并不幸福。

③你生病了，吃了一些中药。

- 你知道自己为什么吃（有意识）；
- 中药的苦味让你感到难受（不快乐）；
- 事后你的病情得以恢复（价值）

即使在当时的体验很痛苦，但事后的痊愈在价值判断中会让一些人觉得“这段经历可以说是幸福的”

（补充一点，在有意识的条件下，我们一般不会进行“没有价值”的事情，①中的价值是情绪价值）

同时，在吃东西时，我们是不是都消耗了“味觉”等其他感觉的注意力？没错，这便是产生幸福的“代价”：我们向其中投入了注意力，但“不幸福”时，我们的注意力便没有获得回报。

从上面的例子便能体会到：我们现在的目标就是减少【无意识】下注意力的消耗，让这个资源更多的投放在【有意识】产出的【幸福】中。

## 六。被无限放大的本性：幻想

我发现自己有时候一天70%的时间都在幻想。

幻想（尤其是过度幻想），是一种“低成本、高预期、内循环”的注意力占用形式。

先给幻想一个**模型内定义**：

幻想 = 注意力在缺乏外部输入的情况下，被“内部预期生成器”持续占用的状态。

之前提到，注意力是**有限资源，而**注意力的流向有两个出口：

① 外界（行动 / 反应）② 内心（记忆 /模拟 /回忆）。

然后补充两个前提：

①快乐来自于结果 ≥ 预期；②回忆、梦境、想象 也能触发接近真实体验的多巴胺反应

 那么幻想的特殊之处在于：它几乎不需要现实世界的摩擦成本，却可以直接制造“预期被满足”的假象。这是它危险、但又极具控制力的地方。

**为什么“过度幻想”会系统性地破坏注意力？因为绕过了「现实反馈」这一关**

在原本的现实循环中：期望 → 投注注意力 → 行动 → 结果 → 快乐 / 修正期望

但幻想构建的是一条**短路循环**：期望 → 注意力 → 内部模拟 → 快感 → 强化期望

这里发生了一件极其关键的事：**注意力获得了多巴胺回报，但没有任何现实结果作为锚点。**

这会导致两个后果：

**Ⅰ注意力学会了：“不去现实，也能拿到奖励”：**现实世界的任务（学习、行动、社交）因为反馈慢、风险高，**在注意力竞争中直接失败——幻想是“无意识状态”里，最精致、最隐蔽的一种注意力消耗。**

**Ⅱ幻想会抬高预期，却不提高现实能力：**快乐发生在「结果 ≥ 预期」而过度幻想的核心副作用是：

它**不断制造高分辨率、高情绪强度的“结果图景”，**但这些“结果”并未经过现实行动的筛选，于是出现结构性失衡：

| 层面 | 变化 |
| --- | --- |
| 预期 | 被幻想无限拔高 |
| 行动能力 | 没有同步提升 |
| 现实反馈 | 必然频繁低于预期 |
| 主观感受 | 挫败、逃避、分心 |

 这会反过来**进一步驱动幻想**，因为幻想仍然是唯一一个“能赢的世界”。这与**刷视频 / 打游戏**的描述高度同构，只是幻想，是一个内生的平台型注意力垄断者。

### 幻想侵占的是「最可贵的注意力形态」

在之前的模型里，其实隐含区分了三种注意力状态：

1. **无意识反射型注意力**（刷视频、条件反射）
2. **有意识但外部导向的注意力**（学习、行动）
3. **有意识但内部导向的注意力**（反思、构建意义）

而**过度幻想**最危险的地方在于：它伪装成第 3 种，但运作逻辑却更接近第 1 种。

表面上在“思考”“想未来”“构建自我叙事”，实际上注意力在做的是：

高频情绪刺激，低现实约束，零价值产出

从“幸福”定义看：有快乐，有价值整合 ，有意识 （因为你的意识并未真正选择它，而是被动产生）所以它既不幸福，也不自由——过度幻想并不是注意力的自由，而是注意力在失去现实锚点后，对低成本多巴胺的一次自我殖民。

尤其在东亚文化圈中，我们面对着下面的困境：

我们处在一个**三重叠加区**。

**历史叙事被高频调用（集体怀旧 / 民族复兴叙事）→ 放大长期幻想层**

**平台化娱乐高度发达（短视频、情绪内容）→ 放大即时多巴胺层**

**个体现实试错空间收缩（高成本失败）→ 压缩中层行动空间**

在我的模型里，这意味着：

顶层幻想很强；底层刺激很强；**中间那层“枯燥但可推进的现实行动”被挤没了**

如此的结果便是：一边不允许你成为虚无的人，一边又让你越来越难成为现实中的人。

社交媒体上的我发现，很难找到工程师：我和他们的层次错位了。或者说注意力的投注方向是不一样的。我们对工程师的印象就是：将他人的幻想建造成现实；更确切地说其中还有设计师，负责让幻想→方案，而工程师对应方案→现实。他们能够让注意力集中于工作的流程，还能从中获得多巴胺的快乐。我在社媒上接触的绝大多数都是表演者；他们从观众的目光获得快乐，他们的注意力能天然的集中在评论区对他们的赞美或批判当中。因此，我对自己的期待，在大众的社交媒体平台上，自然而然地变成了视频中展现的样子，同时还在东亚文化圈中不断徘徊；我越来越难离开屏幕，越来越不由己地幻想。

## 七。马斯洛的金字塔：我为什么认为这会是一次工业革命

在第四部分，我重点分析了一下对短视频和游戏来说，AI的到来可能会优化“注意力”的浪费。我的核心概念有如下几点：无意识、注意力、优化、增加幸福。我会以这种“个体角度中有限资源的优化”的方式来描述前三次工业革命。并且我引入了马斯洛的需求层级：它们并不是一一对应的，而是直觉上相互可解释的。

回顾历史，

第一次工业革命的核心创新是通过**机械化生产**减少了人力资源的低效消耗，改变了传统的手工业模式，解放了人们在**生存/安定**层面的需求。

- **核心机制**：劳动力从繁重的体力劳动中解放出来，人类的生产效率大幅提升，生存问题逐步得到解决。
- **马斯洛对应**：这对应着马斯洛的**生理需求**，也就是人类最基础的生存和安定的需求得到了更高效的满足。

站在**主观视角**来看，第一次工业革命通过解放低效劳动，帮助人们从“无意识”的劳动力消耗中脱离出来，进而获得了更多有意识选择工作的空间，从而接触到了更多的生活**幸福感**。

第二次工业革命进一步通过**能源和大规模生产技术的进步**，减少了资源的浪费，解放了人们对**秩序/稳定**需求的追求。

- **核心机制**：这一阶段的革命不仅提高了生产力，还通过**大规模生产、标准化和工业化**，让物资供应变得更加充足和稳定，解放了人类在生活秩序和资源分配上的焦虑。
- **马斯洛对应**：这一阶段解放了人们的**安全需求**，即生活中对资源、稳定、秩序的渴望得到满足。

从**主观视角**来看，第二次工业革命通过降低资源获取的难度和不确定性，帮助个体减少了对**稳定性和秩序**的焦虑，进入了更具**有意识选择**的生活状态。这让个人不再被生存的紧迫感所困扰，而是有了更多心力去追求自我完善和幸福的可能。

第三次工业革命——**信息技术的爆发**，通过**计算机、互联网和数字化**，大大提高了信息流通的效率，减少了**信息获取的无效损耗**，解放了人们对**归属/社交**需求的关注。

- **核心机制**：信息获取从传统的线性方式转向了更加即时、广泛的网络化，全球化的信息交换使得个体与社会之间的联系变得更加紧密，人们能够更容易接触到同好、朋友、社群，并满足自我认同和归属感的需求。
- **马斯洛对应**：这一阶段解放了人类的**社交需求**，不再需要通过面对面的交流才能建立联系，社交圈的拓展变得更加灵活和即时。

从**主观视角**来看，第三次工业革命通过信息流的高效传递，减轻了人们在信息获取上的压力，人们开始有更多时间和能力去筛选有价值的信息，选择真正能带来自我提升和幸福感的事物。这种“信息自由”使得个体在社会中找到了自己的位置，并能够与志同道合的人群建立联系，进而获得更强的归属感。

站在今天的视角，**注意力的掌控**和**有意识的自我选择**正成为新的解放方向。随着信息时代的飞速发展，过度的社交媒体、视频推荐、娱乐化内容等吸引了大量的注意力，人的注意力逐渐变得“稀缺”。因此，**注意力革命**是第三次工业革命的延续，它解放了个体对自我认知、价值选择和自我掌控的需求。

- **核心机制**：在这个阶段，科技不仅仅帮助人们获取信息，更开始帮助我们**控制自己的注意力**。通过AI技术、个性化推荐等手段，我们可以主动选择我们的信息源、社交圈、娱乐方式，从而获得更多的**自我认知**和**自我提升**的机会。
- **马斯洛对应**：这一阶段将对应到**自尊/自信需求**，当人们能够控制自己的注意力、选择信息、掌握自我时，正是走向更高层次自我探索的起点。

**从主观视角**来看，注意力的掌控意味着人类能够在意识层面选择自己的生活，而不仅仅是被动反应于外界刺激。这种自主性和选择权让我们可以真正地去**探索自我**，找到自我定位，并通过“主动”选择与“有意识”的思维方式实现真正的幸福——当人类掌握了注意力，并开始有意识地调配自己的精力时，他们能够主动塑造自己的世界观和人生观，从而提升**自尊**和**自信**。

### 

从工业革命到信息革命，再到注意力革命，每一轮技术变革都逐步解放了不同层次的需求：从生存到安全，从社交到自尊，**每一次“解放”都带来了人类幸福感的进一步提升**。正如**马斯洛需求理论**所示，人的需求从基础的生存、安全，到更高层次的归属、尊重，一直是递进的。工业革命从“减少无意识的消耗”开始，逐步将人类推向了“有意识”地追求幸福的阶段，而**注意力革命**则是最终实现自我认知、主动选择和自我探索的关键步骤：试想当人们实现了对“注意力”的掌控，这意味着我们能用更多的注意力去主观能动地探索世界，而这个步骤正是“自我探索”的前提，人一旦认识了世界，就认识了自己，就能确立自我是怎么存在于社会的，如此才可能达到真正的“自尊、自信”。

最终，这一切都指向了一个目标：我们始终在追寻一个更高层次的幸福，而这个过程不仅仅是物质层面的改善，更是我们意识和选择的自由带来的内心升华。

## 八。马克思唯物史观能否解释这个变革

马克思的唯物史观认为，**生产力**和**生产关系**是社会发展的基础，历史是由社会经济基础（物质生产条件）和上层建筑（法律、政治、意识形态等）之间的矛盾推动的。在马克思主义的框架下，社会的变革通常源自于生产力的发展与现有生产关系之间的矛盾，这种矛盾推动着社会的进步。

站在“生产力”的角度，“注意力革命”可以理解为一种由信息技术和数字化生产力驱动的变革。随着信息技术、互联网、社交媒体和大数据的飞速发展，生产力达到了一个前所未有的高度，尤其是在信息的获取、传播和处理能力上。互联网、大数据的高度成熟，信息流通早已非常高效与广泛。平台化的经济模式（如Facebook、YouTube、TikTok等）极大地增强了信息的流动性，极大地改变了信息生产和消费的方式。同时，生产力不仅限于物质生产，还包括信息生产和消费。企业通过算法、数据挖掘、推荐系统等技术手段控制和优化注意力的分配，创造了新的“信息劳动”形式，其中注意力本身成为了一种可量化的资源：在传统工业社会，生产关系决定了生产资料和劳动力的关系，而在现代数字经济中，生产关系的核心转向了“内容”与“用户注意力”的关系：用户通过消耗“注意力”，将“内容(信息)”转化为内在的“经历/回忆”这一“精神财富”。

平台公司（如Google、Facebook、TikTok）利用用户数据和注意力来推动广告变现，这形成了一种新的资本主义关系，其中“用户注意力”成为了【生产资料】。这些平台通过推荐算法、广告收入等手段深度嵌入用户的日常生活，改变了个人与社会的互动方式，直接决定了人们的信息流动和社会互动模式。这种生产关系不是通过物质产品交换，而是通过信息和注意力进行交换。用户的注意力被视为平台“生产”的一部分，通过**算法优化和用户行为分析**来提高效率和收益。

马克思认为，生产力和生产关系的变化会导致社会矛盾的变化。在“注意力革命”中，这种矛盾主要表现为：

**阶级分化的加剧**：随着平台资本对用户注意力的集中控制，出现了新的“信息不平等”和“数字贫富差距”。大公司和技术平台通过算法精细化管理垄断了用户的注意力，进一步主导用户的消费行为，获得了巨大的经济回报，而普通用户和创作者则往往处于平台的被动位置，注意力的过度消费和信息的碎片化让他们失去了本可以按照意愿分配注意力、从而获取“财富”的机会。

**劳动者的异化**：用户在平台上不断“刷屏”或消费信息的过程，实际上是被异化的劳动过程。人们的注意力成为了一种劳动形式，但并未获得应有的回报；相反，平台公司通过“注意力掠夺”获得了利润，而用户则丧失了对自己注意力的控制。我们能关注比以往任何时候都丰富的内容，但我们比以往任何时候都更缺少注意力的自主可控性。

从马克思的唯物史观出发，“注意力革命”不仅仅是技术和经济层面的变革，更是由社会的**生产力发展**和**生产关系变化**所决定的。在信息技术高度发展的背景下，平台逐步掌控了用户的注意力，这种对注意力的生产、占有和消费，实际上揭示了新的社会矛盾，尤其是信息不平等、注意力掠夺以及个体自我掌控的缺失。

然而，随着技术的进一步发展，尤其是AI和去中心化技术的应用，**注意力革命**有望逐步解决这些矛盾，使个体从“注意力奴役”中解放出来，进而进入更加平等、自由的数字社会，推动社会向着更高层次的幸福与自由进化。

## 九。一群被注意力不平等摧毁的幻想者

ADHD是最渴望“注意力”自由的，正如从前的奴隶对劳动、工人对大规模技术、学生对信息，都是在最长期的被垄断中，成为了最先一批想到、最深切地感受到工业革命效果的群体。

现在的国内互联网论坛上有无数人声称自己是ADHD，其中大部分是自测的，包括我。但在查阅资料与询问专业人士后，我发现实际并非如此：目前的科研成果表明ADHD 更像是大脑**处理注意力与行为控制的系统性差异**，而不是由某种外部行为引发的后天疾病。那为什么很多人表现出了类似的症状呢？我认为是当前我们面对的的注意力环境，导致了我们的注意力高度分散且难以控制。

有一个比较简单的自测：在面对最高强度刺激的活动时（例如赌博、游戏、性），如果能持续进行很久而且很难感到厌倦，甚至废寝忘食，这种似乎不是典型ADHD所能坚持的。关于【高强度刺激】我想具体说明一下，我所指的实际上是最强烈、最显著的【注意力及时反馈】。这些活动强有力地吻合大脑的底层代码，让你在哪怕产生自我监督的苗头时也能夺走潜意识领导的实际行动。因此，从结果上来看，我们变成了“很容易分心”的人，实际上是“很容易从低反馈走神到高反馈活动的人”，也就是“大脑默认网络与自我意识想要做的事情不一致”的境地。
有可能，我并非ADHD，我只是一个被注意力不平等摧毁的人。对我而言，现实任务反馈慢 + 噪音多 + 情绪成本高，太难了；而幻想：即时反馈 + 完全可控 + 无失败，基本没有门槛。于是我潜意识可能已经迅速完成了一次“最优策略更新”：把注意力迁移到内部模拟世界。于是这便成为了【懒惰/拖延】，而确切来说，我认为是在一个注意力市场中，被迫选择“收益率最高”的资产。

## 十。新的自由和幸福在哪里

在前面我们提到了“幸福”，从最开始的简单等同于“快乐”，到后来与“意识”相结合，然后设想了与“工业革命”的关联性：不难看出，这个“幸福”是没有终点的，是随着人类发展不断变化的。它的设定似乎非常像“自由”，而其所指又有所区别：因为自由更多是权力规范之外人的意志所向，幸福是人在主观能动内的追求方向。它们在某些层面有部分重合，比如“注意力”被解放时，我们实现了该方面的自由并达到幸福。所以很多时候我们可以沿着“自由”的线索找到“幸福”的一角。我们来从自由的角度想象一下：当“注意力”得到彻底解放后，我们能前所未有地专注，无论是完成工作、证明猜想、研究问题还是表达思想，AI的存在能分摊掉绝大部分无意识的机械式操作；在那之后，有一个比较普遍的想法是【创作】：届时可能会存在【创造力】的垄断，这经常在科幻小说中出现，也比较符合沿着马斯洛金字塔往上的结果；也有可能是【历史】，互联网毫无疑问会产生越来越多的幻觉冗杂，无论来自人类还是AI都会如此，此时的真正的知识可能会变得稀缺；还有可能是【情感】，如果智能体进化出了情感，毫无疑问它会无限趋近于真实人类……无论如何，第四次的革命肯定也会向着更高层级“幸福”做出改进。

同时我延伸开地想象了一下：每一次革命，实际上都伴随着社会阶层的转型，或者说人与人之间关系的进化；我猜测，AI的这一次也会如此。人们会变得更加平等，可能是在“行动力与知识兑现能力”层面实现的。

## 十一。新的机会，时间，可能的第五次革命

之前提到的用AI优化“无意识”用户行为的短视频推荐算法

已经大规模普及的对话模型比较直观（优化了用户搜索信息和比对的“无意识”，当然是日常生活级别的，专业水平和高度个人化目前仍未实现）

有一个点：历史对于生产者而言，劳动力=时间；=时间；信息的传收=时间；注意力=时间。在人类实现永生前，时间永远是量化生产活动的最底层因子。
